# 获取网站 Favicon

!> 根据URL地址获取网站的图标



## API文档

**接口地址：**`https://api.nie.ge/api/favicon`

**返回格式：**图片

**请求方式：**get

**请求示例：**`https://api.nie.ge/api/favicon/nie.ge.png`

**请求参数说明：**

| 名称 | 必填 | 类型   | 说明                                                         |
| :--- | :--- | :----- | :----------------------------------------------------------- |
| url  | 是   | string | 需要获取图标的URL地址，如：nie.ge，确保URL能够正常打开 不需要 http(s):// ，且结尾必须填 .png |

**返回参数说明：**

| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| 无   | 无   | 无   |

**返回示例：**

```
返回网址图标
```
